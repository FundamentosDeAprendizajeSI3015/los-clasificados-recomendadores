# Datasets Used for Validation

This repository includes multiple datasets that were used to validate the proposed algorithm.  
Each dataset contains a different number of features and samples, allowing for a diverse evaluation of the method.  
The datasets are commonly used for classification tasks and are sourced from well-known repositories such as UCI and OpenML.

## Dataset Overview

The following table summarizes the key characteristics of the datasets:

| **Dataset**          | **Number of Features** | **Number of Samples** | **Reference** |
|----------------------|----------------------|----------------------|--------------|
| Sonar               | 7                    | 209                  | [Sonar Dataset](https://doi.org/10.24432/C5JP4S) |
| Diabetes            | 8                    | 768                  | [Diabetes Dataset](https://doi.org/10.24432/C5XW26) |
| Heart Disease       | 13                   | 303                  | [Heart Disease Dataset](https://doi.org/10.24432/C5RP4T) |
| Australian Credit   | 14                   | 691                  | [Australian Credit Dataset](https://doi.org/10.24432/C52P4G) |
| German Credit       | 20                   | 1000                 | [German Credit Dataset](https://doi.org/10.24432/C5ZP4H) |

## References
All datasets are publicly available and can be accessed through the provided links.
